Kick Start Your Development in the Cloud

Coding WebIDE 是 Coding 自主研发的在线集成开发环境 (IDE)。用户可以通过 WebIDE 创建项目的工作空间, 进行在线开发, 调试等操作。同时 WebIDE 集成了 Git 代码版本控制, 用户可以选择 Coding、GitHub、BitBucket、Git@OSC 等任意的代码仓库。 WebIDE 还提供了分享开发环境的功能, 用户可以保存当前的开发环境, 分享给团队的其他成员。

    基于 Container 技术

    Container Technology

    • 无需休眠 急速启动

      Container 技术让系统的预热时间从分钟级降到秒级。

    • 保存环境 方便分享启动

      配置好的开发环境可以快速的保存，配置工作版本化。同时也对于同项目的其它用户可见，方便的分享环境配置的成果。

    全功能 Web Terminal

    Web Terminal

    支持 Unicode 宽字符

    完美支持宽字符的输入和显示。

    完善的 Web 终端模拟器

    兼容标准的终端模拟器，支持 Bash，Zsh，Fish 等各种 Shell，支持自定义 PS1。默认集成了 Zsh 和 oh-my-zsh 以达到更好的输入支持。

    提供 root 权限

    默认用户 sudo 免密码，用户可以自由安装偏好的软件包。

    便捷的运行时预览

    Easy Runtime Preview

    临时访问地址

    快捷生成唯一的临时 URL 地址，方便 HTML5 开发迅速预览成果。

    TCP 端口发现

    对于后端启动 TCP 服务，后端会自动生成临时访问地址，并可以在 terminal 内通过 open 命令弹出预览界面。

    实时预览的内容编辑

    Editor with Live Preview

    内置了 Markdown 和 HTML 编辑器，可以实时预览编辑效果。

    更多特性

    More Features

    语法加亮

    支持 100 多种编程语言的语法加亮
    可以方便地切换不同的代码着色风格

    代码补全

    支持常见开发语言的代码输入提示和自动补全

    主题切换

    对于不同的主题颜色会自动匹配合适的代码着色风格

    分割视图

    流行的视图布局 边栏支持收缩
    边界区域支持最大化 可以无限分割编辑区

    VIM／Emacs 模式

    可以开启 VIM / Emacs 模式
    兼容响应模式的输入快捷键

    预览工具

    内置了图片预览、Markdown 预览、HTML 预览等开发预览工具
